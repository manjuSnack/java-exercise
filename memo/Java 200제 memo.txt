🍘 e(fx)clipse 플러그인 설치하기

Help -- Eclipse MarketPlace -- Find -- javafx -- e(fx)clipse Install

: e(fx)clipse 3.8.0으로 설치

🍘 e(fx)clipse를 이용하여 JavaFX 프로젝트 만들기

File -- New -- Other -- JavaFx -- JavaFx Project -- Project명 작성 / Next -- 아래 내용을 참고 후 Finish

Application type
: Desktop 혹 Mobile 선택

Package Name
: 패키지명을 입력

Language
: FXML 등 이용

Root-Type
: 기본 값은 BorderPane, 나중에 변경이 가능하다

File Name
: FXML의 이름을 입력

Controller Name
: 화면과 이벤트 및 로직을 연결할 클래스 이름을 입력한다.

만약, 에러가 발생 시 SDK에 대한 Libraries를 추가하여야 한다.
